Returns canvas size.
## Usage
```
$canvasSize[canvas;property]
```

## Parameters
| Field | Description | Type | Required |
| --- | --- | --- | --- |
| canvas | The canvas name. | string | true |
| property | The canvas size property. | "width" \| "height" | true |
