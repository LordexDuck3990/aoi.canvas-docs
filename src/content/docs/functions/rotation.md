Sets canvas rotation.
## Usage
```
$rotation[canvas;angle]
```

## Parameters
| Field | Description | Type | Required |
| --- | --- | --- | --- |
| canvas | The canvas name. | string | true |
| angle | The rotation angle. | number | true |
