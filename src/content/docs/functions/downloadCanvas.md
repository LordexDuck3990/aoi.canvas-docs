Downloads the canvas.
## Usage
```
$downloadCanvas[canvas;path]
```

## Parameters
| Field | Description | Type | Required |
| --- | --- | --- | --- |
| canvas | The canvas name. | string | true |
| path | The download path. (file name and extension too) | string | true |
